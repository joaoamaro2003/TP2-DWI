## Descrição do projeto

PokéGuess é um jogo de adivinhação de Pokémon baseado na PokéAPI.  
O jogador tenta descobrir o Pokémon através de diferentes modos de jogo, ganhando pontos por respostas corretas.

O jogo inclui sistema de vidas, pontos, streak, timer e histórico de partidas.
